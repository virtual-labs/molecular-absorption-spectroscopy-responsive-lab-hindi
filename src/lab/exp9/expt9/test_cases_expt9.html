  स्पेक्ट्रोफोटोमीटर पर चमकती शुरू करो।    
बीकर पर क्लिक करें बीकर मेज पर चला जाता है अच्छा कर रहा है अच्छा कर रहा है
फ्लास्क पर क्लिक करें फ्लास्क मेज पर चला जाता है। अच्छा कर रहा है अच्छा कर रहा है
फ्लास्क पर क्लिक करें फ्लास्क से तरल डालना दिखाया गया है अच्छा कर रहा है अच्छा कर रहा है
  बीकर में। बीकर फिर लौट आता है

पिपेट पर क्लिक करें पिपेट मेथिलोरेंज समाधान के लिए ले जाता है और अच्छा कर रहा है अच्छा कर रहा है
  तरल चूसता है।    
पिपेट पर क्लिक करें पिपेट बीकर में चला जाता है। अच्छा कर रहा है अच्छा कर रहा है
पिपेट पर क्लिक करें पिपेट समाधान जारी करता है, शेल्फ में वापस जाता है, अच्छा कर रहा है अच्छा कर रहा है
  बीकर की सामग्री लाल हो जाती है।    
Micropipette पर क्लिक करें पिपेट बीकर में चला जाता है। अच्छा कर रहा है अच्छा कर रहा है
Micropipette पर क्लिक करें पिपेट बीकर से तरल चूसता है। अच्छा कर रहा है अच्छा कर रहा है
क्युवेट पर क्लिक करें Cuvette तालिका में नीचे जाता है। अच्छा कर रहा है अच्छा कर रहा है
Micropipette पर क्लिक करें माइक्रोप्रिपेट क्युवेट की ओर बढ़ता है। अच्छा कर रहा है अच्छा कर रहा है
Micropipette पर क्लिक करें Micropipette समाधान जारी करता है, micropipette चलता है अच्छा कर रहा है अच्छा कर रहा है

क्युवेट पर क्लिक करें Cuvette स्पेक्ट्रोफोटोमीटर की ओर बढ़ता है, एक और क्युवेट अच्छा कर रहा है अच्छा कर रहा है
  प्रदर्शित किया जाता है स्पेक्ट्रोफोटोमीटर।    
स्पेक्ट्रोफोटोमीटर पर क्लिक करें स्पेक्ट्रोफोटोमीटर का ढक्कन बंद हो जाता है। अच्छा कर रहा है अच्छा कर रहा है
